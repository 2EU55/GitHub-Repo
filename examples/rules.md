# 2026年Q1 销售奖金计算规则（示例）

## 1. 数据范围
- 统计周期：2026-01-01 至 2026-03-31（含首尾）
- 数据来源：销售业务明细表

## 2. 指标定义
- 总营收 = Σ(消费金额)
- 产品系数：根据“产品名称”映射系数
- 有效业绩 = Σ(消费金额 × 产品系数)
- 应发奖金 = 有效业绩 × 奖金比例
- 排名：按应发奖金降序排序

## 3. 产品系数规则
- 产品A：1.00
- 产品B：0.90
- 产品C：0.80
- 未配置产品：系数默认 1.00

## 4. 奖金比例规则
方案A（固定比例，默认）：奖金比例 = 2.5%

方案B（分档比例，可选）：
- 有效业绩 < 100万：1.5%
- 100万 ≤ 有效业绩 < 300万：2.0%
- 300万 ≤ 有效业绩 < 500万：2.5%
- 有效业绩 ≥ 500万：3.0%
